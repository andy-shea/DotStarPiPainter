# DotStarPiPainter
Light painting project for Raspberry Pi + DotStar strips

Tutorial is here: https://learn.adafruit.com/dotstar-pi-painter

## Wiring

Black: Ground

Yellow: Clock In (SCK on Adafruit M4 Express)

Green: Data In (MO on Adafruit M4 Express)

Red: 5V
